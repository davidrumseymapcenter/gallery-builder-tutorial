## Open the Gallery Builder tool
- [https://davidrumseymapcenter.github.io/set-builder/](https://davidrumseymapcenter.github.io/set-builder/){:target="_blank"}

Keep the tool open in one browser tab and this tutorial open in another. 
<hr>
## Copy a manifest

- We have provided you with a few manifests to start out. 

- Choose one of the links below and copy it using the clipboard icon in the upper right corner of the box.

```html
https://www.davidrumsey.com/luna/servlet/iiif/m/RUMSEY~8~1~296148~90067635/manifest
```

```html
https://purl.stanford.edu/ht369zy9186/iiif/manifest
```

```html
https://iiif.archive.org/iiif/map198204401/manifest.json
```
<hr>
## Add the manifest to your gallery
- In the IIIF Gallery Builder tool, paste the manifest into the top field and click *“Add Manifest(s)”*

- If the item contains multiple pages, you will see a pop-up window where you can select the desired page or pages. Choose the pages to include and click *"Add Selected Pages"*

- A card or cards should appear below the Input Panel containing the thumbnail and metadata for the item you just added.

{: .new}
***Add the remaining items following this same workflow.***
