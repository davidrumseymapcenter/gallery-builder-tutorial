# Save your work to your computer

## Name your collection

- Choose a title for your gallery and type it in the field that says "Enter gallery name".
  
>{: .highlight }
If you forget to choose a name the tool will autogenerate one with today's date

## Save to your download directory

- Click the Save Gallery button

- Navigate to your downloads folder (this may be your desktop) and find the file you just saved. It will have the title you chose with the suffix "-gallery.json"

# Share the file with others

Push files to Google Drive shared google drive

Once completed, everyone should download  someone else’s file and open it

## Send your JSON file via email 
- After saving your gallery locally, attach your JSON file to an email

- In the body of the email, paste the link to the webapp: https://davidrumseymapcenter.github.io/set-builder/

- Tell the recipient to download the file to their desktop and open it using the “Load Gallery” button in the app
